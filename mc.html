Lectures

  1. Introduction
    • motivation for model checking
    • difference with simulation and testing
    • glance at model checking with temporal logics
  2. Modeling by labeled transition systems
    • modeling by Labeled Transition Systems (LTSs)
      • exections, traces,
      • non-determinism
      • examples
  3. Linear-Time Behavior & Properties
    • path fragments and paths
    • invariant, safety, and liveness properties
    • every LT-property is the intersection of a safety and a liveness property
    • fairness
  4. Linear Temporal Logic (LTL)
    • syntax and semantics
    • derived modalities, equivalence of formulas
    • interpretation in a transition system
  5. Linear Temporal Logic (LTL)    [notes 2024 (pdf)]
    • model checking LTL formulas
      • express properties by (generalized) Büchi automata
        • examples of translations of LTL formulas ( ◯a    and   a U b ) into generalized Büchi automata
      • model check transition systems and properties via product automata
      • complexity of this model-checking procedure
    • fairness in LTL
  6. Computation Tree Logic (CTL)    [notes 2024 (pdf)]
    • syntax and semantics
    • examples
    • model checking
    • expressibility difference compared to LTL
  7. CTL continued & Outlook    [notes 2024 (pdf)]
    • CTL+
    • CTL*
    • Venn diagram comparison of the expressibility of LTL, CTL, CTL+, CTL*, and the μ–calculus
    • Outlook, either of:
      • partial model checking    [notes 2024 (pdf)]
        • based on the research paper Partial Model Checking and Partial Model Synthesis in LTL Using a Tableau-Based Approach (link) by Serenella Cerrito, Valentin Goranko, Sophie Paillocher at FSCD 2023 on partial model checking
      • model μ–calculus
        • based on the book chapter The mu-calculus and model-checking (link) by Julian Bradfield and Igor Wałukiewicz
